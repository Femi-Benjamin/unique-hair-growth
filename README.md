# Welcome to Dhee’s Royal Growth

## Project info

**URL**: https://dhee-s-royal-growth.vercel.app/

Dhee’s Royal Growth is a product landing page for Dhee’s Organic Hair Treatment — a natural hair care solution designed to nourish the scalp, promote stronger and thicker hair growth, reduce hair fall and dandruff, and enhance overall hair health using herbal and organic ingredients. The site showcases the product’s benefits, features and usage for customers interested in premium hair growth treatment products.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
